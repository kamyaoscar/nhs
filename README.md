# Namungoona High School - Official Website

Welcome to the official source code repository for the **Namungoona High School** homepage. This project serves as the digital front door for our institution, providing parents, students, and stakeholders with vital school information, leadership details, and core values.

🌐 **Live Website:** [Click here to view the live site](https://kamyaoscar.github.io/nhs/)

---

## 🏫 About the School

* **Name:** Namungoona High School
* **Established:** 1999
* **UNEB Center Number:** U1256
* **Motto:** *Strive to Excel*
* **Location:** 5 Kilometers along Hoima Road (100 meters off Nakibinge Stage), Kampala, Uganda
* **School Leadership:** Hon. Ssemaganda Ronald (Director)

### Vision & Mission
* **Vision:** To be a leading center of academic excellence in secondary education.
* **Mission:** To impart affordable, qualitative, and skilled knowledge through viable and reliable means for sustainable learners.

---

## 🛠️ Project Structure & Technologies

This website is built using clean, lightweight, semantic front-end code with zero external dependencies to ensure fast loading times on mobile devices across Uganda.

* **`index.html`** - Contains the semantic structure, leadership section, and a 290-character Google-optimized SEO meta description.
* **`style.css`** - Custom responsive styling using CSS flexbox/grid variables and an elegant corporate school color palette (School Blue & Prestige Gold).
* **`script.js`** - Lightweight, vanilla JavaScript handling mobile responsive navigation toggles.
